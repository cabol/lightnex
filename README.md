## 🚀 Installation

Add `lightnex` to your dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:lightnex, "~> 0.1"}
  ]
end
```

For more detailed information, see the
[getting started guide][getting_started] and
[online documentation][docs].
